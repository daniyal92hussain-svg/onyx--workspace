**Onyx** is a high-end, luxury project management ecosystem designed for elite agile teams. Moving away from generic "ProManage" vibes, Onyx offers a "Glassmorphism" aesthetic with a user-first philosophy—where every native HTML element has been replaced with custom-crafted, animated widgets.

---

## Design Philosophy
"Software shouldn't just be functional; it should be an experience." 

Onyx was built to eliminate the 'boring' from management. No native dropdowns, no janky transitions—just pure, buttery-smooth flow.

---

## Key Features

### Cinematic Experience
* **Premium Splash Screen:** A blurred, animated entry with a sliding gemstone logo and fade-in welcome.
* **App Reveal:** Smooth mounting animation with scale and opacity transitions.

### Linked CRM (Client Hub)
* **Direct Association:** Clients are contextually linked to their respective projects.
* **Activity Logs:** Maintain a professional history for every client with manual note-taking and automated system logs (e.g., status changes).
* **Smart Filtering:** View Active, Completed, or All projects per client using custom-built dropdowns.

### Advanced Workspaces
* **Interactive Kanban:** A fully functional task board with drag-and-drop capability.
* **Project Analytics:** Live health tracking, progress bars, and team point distribution.
* **Subtask Systems:** In-depth task breakdown with real-time checklist progress.

---

##  Technical Stack

* **Core:** React 18 (Vite)
* **Styling:** Tailwind CSS (Utility-first, responsive design)
* **Animations:** Framer Motion & CSS Keyframes
* **Data:** Persistent LocalStorage for real-time CRUD operations without a database.
